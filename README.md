# Portfolio Website

This is a personal portfolio website for showcasing projects, research, and professional background.

## Project Structure

- `src/assets/css/style.css`: Contains styles for the website, including layout, typography, and responsive design.
- `src/js/main.js`: Includes JavaScript code for the website, implementing features such as automatic year display and other interactive elements.
- `src/pages/about.html`: HTML structure and content for the "About" page.
- `src/pages/index.html`: Homepage of the website with introductory content and links to other pages.
- `src/pages/projects.html`: Outlines the "Projects" page, showcasing various projects and their descriptions.
- `src/pages/research.html`: Presents the "Research" page, detailing research interests, publications, and focus areas.

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/ashioyajotham/portfolio-website.git
   ```
2. Navigate to the project directory:
   ```
   cd portfolio-website
   ```
3. Install dependencies:
   ```
   npm install
   ```

## Usage

To start the development server, run:
```
npm start
```

Open your browser and go to `http://localhost:3000` to view the website.
